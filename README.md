# RAG From Scratch

A Retrieval-Augmented Generation pipeline built from first principles using: 
-PyMuPDF
-Sentence Transformers
-FAISS
-Hugging Face Transformers

## Architecture

PDF
- Chunking
- Embeddings
- Vector Store
- Retrieval 
- LLM Generation

## Features

- PDF Ingestion
- Text chunking with overlap
- SBERT embeddings
- FAISS similarity search
- COntext-aware question answering
